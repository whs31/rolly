rolly::source_location struct
#include <rolly/source_location.h>

Provides information about the location of the current source file.

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

Class provides information about the location of the current source file. This class is a port of C++20 std::source_location. Example usage:

auto main() -> int {
  log::debug("I am called from {}, {}:{} in file {}!",
    rolly::source_location::current().function_name(),
    rolly::source_location::current().line(),
    rolly::source_location::current().column(),
    rolly::source_location::current().file_name()
  );
  fmt::println("I am called from {}", rolly::source_location::current());
}

Output:

I am called from main, 5:7 in file main.c++!
I am called from main.c++:main 6:7

Note

This implementation can be aliased to std::source_location, if the compiler supports it.
