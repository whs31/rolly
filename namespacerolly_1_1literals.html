ROLLY_SEMVER_CONSTEXPR version rolly::literals::operator""_version(char const* str, std::size_t length)
#include <include/rolly/global/semver.h>

Version literal operator.

Creates a version from a string literal at compile time.

Example usage:

using namespace rolly::literals;

auto constexpr v = "1.0.0"_version;

static_assert(v == rolly::version(1, 0, 0));
