template<typename T>
optional<std::reference_wrapper<T>> rolly::dll::plugin_loader::query_interface_by_type() const

Queries plugin interface by it's type via RTTI.

Template parameters
T Interface type.
Returns Reference to the retrieved interface or none when no such interface is loaded.

Essentially iterates over all loaded plugins and tries to dynamic_cast their pointer to the requested interface.

Warning

This function can be slow, especially when many plugins are loaded. Much better solution will be to query by internal plugin interface name using method query_interface
